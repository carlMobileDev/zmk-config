```
                        ╰───────┴───────┴───────╯   ╰───────┴───────┴───────╯
```

**Homerow Mods** (hold to activate):

| Key | Tap  | Hold   |
|-----|------|--------|
| S   | s    | GUI    |
| D   | d    | CTRL   |
| F   | f    | SHIFT  |
| J   | j    | SHIFT  |
| K   | k    | CTRL   |
| L   | l    | GUI    |

## Lower Layer (left thumb)

Left hand = shifted symbols, Right hand = base symbols (no shift needed for pairs).

```
╭───────┬───────┬───────┬───────┬───────┬───────╮   ╭───────┬───────┬───────┬───────┬───────┬───────╮
│  TAB  │   !   │   @   │   #   │   $   │   %   │   │   ^   │   &   │   *   │   (   │   )   │ BSPC  │
├───────┼───────┼───────┼───────┼───────┼───────┤   ├───────┼───────┼───────┼───────┼───────┼───────┤
│       │   "   │   {   │   }   │   _   │   +   │   │   '   │   [   │   ]   │   -   │   =   │   \   │
├───────┼───────┼───────┼───────┼───────┼───────┤   ├───────┼───────┼───────┼───────┼───────┼───────┤
│       │   ~   │   |   │       │       │       │   │   `   │   \   │       │       │       │       │
╰───────┴───────┴───────┼───────┼───────┼───────┤   ├───────┼───────┼───────┼───────┴───────┴───────╯
```

## Quick Reference

| Action          | How                                |
|-----------------|------------------------------------|
| `:`             | SHFT + `;`                         |
| `"`             | LOWER + `"` (left hand, direct)    |
| `_`             | LOWER + `_` (left hand, direct)    |
| `+`             | LOWER + `+` (left hand, direct)    |
| `~`             | LOWER + `~` (left hand, direct)    |
| `|`             | LOWER + `|` (left hand, direct)    |
| `'`             | Base layer right pinky, or LOWER right hand |
| `\`             | LOWER + `\` (right hand)           |
| tmux prefix     | Hold D (CTRL) + b                  |
| vim command     | Hold D (CTRL) + SHFT + `;`         |
| vim escape      | Top-left ESC                       |
